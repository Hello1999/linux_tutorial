## 💡 学习建议

### 每日学习计划

**上班日（1-2小时）**
- 30分钟理论学习
- 30分钟实践练习
- 30分钟编写笔记/博客

**周末（3-4小时）**
- 1小时复习本周内容
- 2-3小时完成实战项目
- 整理学习笔记

### 学习技巧

1. **动手实践**
   - 不要只看不做
   - 所有命令都要亲自敲一遍
   - 尝试修改参数看效果

2. **遇到问题**
   - 先查man手册：`man command`
   - 参考[故障排查指南](./故障排查指南.md)
   - Google搜索错误信息
   - 在Linux社区提问

3. **建立知识库**
   - 记录常用命令
   - 保存自己的脚本
   - 写技术博客
   - 使用Git管理笔记

4. **定期复习**
   - 每周复习一次
   - 使用速查表快速回忆
   - 重做练习题

5. **参与社区**
   - Stack Overflow
   - Reddit r/linux
   - Linux中文论坛
   - GitHub开源项目

## 🔧 推荐工具

### 学习环境
- **虚拟机**：VirtualBox + Ubuntu 22.04 LTS
- **终端**：Terminator / Tilix
- **编辑器**：Vim / VS Code
- **笔记**：Obsidian / Notion / Markdown

### 实用工具
```bash
# 安装推荐工具
sudo apt install htop tmux tree tldr ncdu
```

- **htop** - 进程监控
- **tmux** - 终端复用
- **tree** - 目录树显示
- **tldr** - 简化的命令帮助
- **ncdu** - 磁盘使用分析
