# Linux常用命令速查表

## 文件和目录操作

| 命令 | 说明 | 示例 |
|------|------|------|
| `ls` | 列出目录内容 | `ls -lah` |
| `cd` | 切换目录 | `cd /etc` |
| `pwd` | 显示当前目录 | `pwd` |
| `mkdir` | 创建目录 | `mkdir -p dir1/dir2` |
| `rmdir` | 删除空目录 | `rmdir mydir` |
| `touch` | 创建空文件 | `touch file.txt` |
| `cp` | 复制 | `cp -r source dest` |
| `mv` | 移动/重命名 | `mv old.txt new.txt` |
| `rm` | 删除 | `rm -rf directory` |
| `find` | 查找文件 | `find /path -name "*.txt"` |
| `locate` | 快速查找 | `locate filename` |
| `which` | 查找命令位置 | `which python3` |
| `tree` | 树状显示目录 | `tree -L 2` |

## 重定向和管道

| 符号 | 说明 | 示例 |
|------|------|------|
| `>` | 输出重定向（覆盖） | `echo "test" > file.txt` |
| `>>` | 输出重定向（追加） | `echo "test" >> file.txt` |
| `<` | 输入重定向 | `command < input.txt` |
| `2>` | 错误重定向 | `command 2> error.log` |
| `&>` | 标准输出和错误重定向 | `command &> output.log` |
| `\|` | 管道 | `ps aux \| grep nginx` |
| `\|&` | 管道（包含错误） | `command \|& tee output.log` |
| `tee` | 同时输出到屏幕和文件 | `command \| tee output.txt` |

## 常用正则表达式

| 符号 | 说明 | 示例 |
|------|------|------|
| `.` | 任意单个字符 | `a.b` 匹配 `axb` |
| `*` | 零个或多个 | `a*b` 匹配 `b`, `ab`, `aab` |
| `+` | 一个或多个 | `a+b` 匹配 `ab`, `aab` |
| `?` | 零个或一个 | `a?b` 匹配 `b`, `ab` |
| `^` | 行首 | `^start` |
| `$` | 行尾 | `end$` |
| `[abc]` | 字符集 | `[aeiou]` |
| `[^abc]` | 非字符集 | `[^0-9]` |
| `[a-z]` | 范围 | `[A-Z]`, `[0-9]` |
| `\d` | 数字 | 等同于 `[0-9]` |
| `\w` | 单词字符 | 等同于 `[A-Za-z0-9_]` |
| `\s` | 空白字符 | 空格、制表符等 |
| `\|` | 或 | `cat\|dog` |
| `()` | 分组 | `(ab)+` |

## 提示

- 使用 `man command` 查看详细手册
- 使用 `command --help` 查看快速帮助
- 使用 `tldr command` 查看实用示例（需安装tldr）
- 使用 `Tab` 键自动补全
- 使用 `Ctrl+R` 搜索历史命令
- 使用 `history` 查看命令历史
- 危险操作前先备份！
- 不确定就先在测试环境试验
