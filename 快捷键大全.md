# Linux快捷键大全

## Bash/Shell快捷键

### 光标移动
| 快捷键 | 功能 |
|--------|------|
| `Ctrl+A` | 移到行首 |
| `Ctrl+E` | 移到行尾 |
| `Ctrl+B` | 向后移动一个字符 |
| `Ctrl+F` | 向前移动一个字符 |
| `Alt+B` | 向后移动一个单词 |
| `Alt+F` | 向前移动一个单词 |
| `Ctrl+XX` | 在行首和当前位置切换 |

### 编辑
| 快捷键 | 功能 |
|--------|------|
| `Ctrl+D` | 删除光标处字符 |
| `Ctrl+H` | 删除光标前字符（退格） |
| `Ctrl+W` | 删除光标前的单词 |
| `Alt+D` | 删除光标后的单词 |
| `Ctrl+U` | 删除从光标到行首 |
| `Ctrl+K` | 删除从光标到行尾 |
| `Ctrl+Y` | 粘贴之前删除的内容 |
| `Ctrl+T` | 交换光标前两个字符 |
| `Alt+T` | 交换光标前两个单词 |
| `Alt+U` | 将光标到单词尾转为大写 |
| `Alt+L` | 将光标到单词尾转为小写 |
| `Alt+C` | 将光标处字母大写 |

### 历史命令
| 快捷键 | 功能 |
|--------|------|
| `Ctrl+P` | 上一条命令（等同于↑） |
| `Ctrl+N` | 下一条命令（等同于↓） |
| `Ctrl+R` | 反向搜索历史命令 |
| `Ctrl+S` | 正向搜索历史命令 |
| `Ctrl+G` | 退出历史搜索模式 |
| `Alt+.` | 插入上一命令的最后参数 |
| `Alt+数字+.` | 插入上第N条命令的最后参数 |
| `!!` | 重复上一条命令 |
| `!$` | 上一命令的最后一个参数 |
| `!^` | 上一命令的第一个参数 |
| `!*` | 上一命令的所有参数 |
| `!n` | 执行历史中第n条命令 |
| `!string` | 执行最近以string开头的命令 |

### 进程控制
| 快捷键 | 功能 |
|--------|------|
| `Ctrl+C` | 中断当前命令 |
| `Ctrl+Z` | 暂停当前命令 |
| `Ctrl+D` | 退出当前Shell（EOF） |
| `Ctrl+S` | 停止输出到屏幕 |
| `Ctrl+Q` | 恢复输出到屏幕 |

### 屏幕操作
| 快捷键 | 功能 |
|--------|------|
| `Ctrl+L` | 清屏（等同于clear） |
| `Ctrl+[` | 等同于ESC |
| `Tab` | 自动补全 |
| `Tab Tab` | 显示所有可能的补全 |

## 实用技巧

### 历史命令技巧
```bash
# 使用历史替换
!100              # 执行历史中第100条命令
!-2               # 执行倒数第2条命令
!!                # 重复上一条命令
sudo !!           # 用sudo重复上一条命令

# 参数替换
^old^new          # 替换上一命令中的字符串
```

### 快速移动技巧
```bash
cd -              # 返回上一个目录
pushd /path       # 保存当前目录并切换
popd              # 返回保存的目录
```

### Readline技巧
```bash
# 在.inputrc中自定义
set completion-ignore-case on    # 补全忽略大小写
set show-all-if-ambiguous on     # 显示所有可能补全
```
